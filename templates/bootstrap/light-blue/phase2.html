Stanton Space Barn Phase 2 Testbed

What is Phase 2?

What you see on the main website is the original Stanton Space Barn. This was built months before the Hangar release, and months before I had any solid idea how the game was going to function. As a result while it is fun to play with, it doesn't accurately reflect the items in game, nor the actual methods of customizing them.

Phase 2 is the next iteration of the Space Barn that will take the ship modding functionality of the current site, marry it with the live game data you see in the DataBanks, and combine all that with a new system that works like it does in game.

Phase 2 isn't finished yet, and it isn't integrated into the main site. But I like to share, so here you can find some links to specific ships that are up and running in the Phase 2 testbed. These ships and the items with them are all pulling from the live game data, so it is 100% accurate to the game. You can't save or share these yet, that functionality is still in work, but enjoy this small sneak peak!
